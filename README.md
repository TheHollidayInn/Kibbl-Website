##Installation
 - Install NodeJs
 - Install Mongo
 - Install yarn - `npm install -g yarn`
 - Install gulp - `npm install -g gulp`
 - Run Gulp Concat - `gulp concat`
 - Run Yarn - `yarn`
 - Run in Dev - `npm run dev`
 - Go to (localhost:3000)[http://localhost:3000]
